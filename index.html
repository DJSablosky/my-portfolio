Serverless Portfolio

This website is a serverless application hosted and deployed with Amazon Web Services. It uses GitHub for source control, IAM for security, Route 53 for DNS, S3 for object storage, and CloudFront for distribution of those objects. In addition, it applies CodeBuild, CodePipeline, and Lambda to build the application, deploy it to the S3 bucket, and notify subscribers with SNS. The front-end of the application was created utilizing the React JavaScript library, HTML, and CSS.
